# Employee Attrition Analysis and Prediction

## Project Overview
This project analyzes employee attrition using HR Analytics, SQL, Python, Machine Learning, and Dashboard Visualization.

## Objectives
- Identify factors causing employee attrition
- Predict employees likely to leave
- Build interactive HR dashboards
- Provide retention recommendations

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- SQL
- Git & GitHub

## Project Structure
- Data
- notebooks
- Sql
- Dashboard
- Screenshots
